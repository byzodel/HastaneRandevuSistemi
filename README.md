# Hastane Randevu Sistemi

Bu proje, C# ve Windows Forms kullanılarak geliştirilmiş bir hastane randevu sistemidir. 
Kullanıcıların branş, doktor, tarih ve saat seçerek randevu almasını amaçlamaktadır.

## Özellikler
- Kullanıcı giriş ekranı
- Branş seçimi
- Doktor seçimi
- Tarih ve saat seçimi
- Randevu onay ekranı
- Hatalı girişler için uyarı mesajları

## Kullanılan Teknolojiler
- C#
- Windows Forms
- .NET Framework

## Projenin Çalıştırılması
1. Bu repository'yi indirin (Code > Download ZIP).
2. ZIP dosyasını bilgisayarınıza çıkartın.
3. Visual Studio ile `.sln` uzantılı dosyayı açın.
4. Projeyi çalıştırın.

## Bağımlılıklar
- Ekstra bir kütüphane veya paket gerektirmemektedir.
- .NET Framework yüklü olması yeterlidir.
